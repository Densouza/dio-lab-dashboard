Dashboard de Vendas - Assinaturas

Este projeto apresenta um dashboard de vendas de assinaturas criado no Microsoft Excel, com o objetivo de visualizar e analisar o desempenho de planos, receitas e comportamentos de clientes.

Objetivo do Projeto

O objetivo é analisar o desempenho das assinaturas a partir de uma base de dados contendo informações sobre clientes, planos contratados e valores pagos.
O dashboard permite acompanhar métricas importantes de forma clara e interativa.

Dados Utilizados

Os dados foram obtidos a partir da aba “Bases”, contendo as seguintes colunas principais:

Coluna	Descrição
Subscriber ID	Identificador único do assinante
Name	Nome do cliente
Plan	Tipo de plano contratado (Ultimate, Core, Standard)
Start Date	Data de início da assinatura
Auto Renewal	Indica se a assinatura é renovada automaticamente
Subscription Price	Valor base do plano
EA Play Season Pass e Minecraft Season Pass	Add-ons opcionais
Coupon Value	Valor de desconto aplicado
Total Value	Valor total pago pelo cliente

Como Reproduzir

Abra o arquivo Dashboard_Assinaturas_Pronto.xlsx no Microsoft Excel.

Vá até a aba “Dashboard” para visualizar os indicadores e gráficos.

Se quiser atualizar os dados:

Edite a aba “Bases” com novas informações;

Atualize as tabelas dinâmicas e gráficos pressionando Ctrl + Alt + F5.
